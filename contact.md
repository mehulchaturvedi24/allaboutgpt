# Contact

Have a question, suggestion, or want to contribute to All About GPT? Get in touch.

---

## Get In Touch

You can reach out via the following channels:

- **GitHub Issues:** [Open an issue](https://github.com/mehulchaturvedi24/allaboutgpt/issues) on the repository for suggestions or corrections.
- **GitHub Discussions:** Share ideas or ask questions in the [Discussions tab](https://github.com/mehulchaturvedi24/allaboutgpt/discussions).

---

## Contribute

This site is open source and welcomes contributions.

- Found a broken link? Open an issue.
- Want to add a resource? Submit a pull request.
- Have a blog post idea? Fork the repo and add a file to `_posts`.

All contributions are welcome and appreciated.
